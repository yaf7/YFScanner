**YFScanner** adalah aplikasi pemindai dokumen berbasis Android yang efisien dan minimalis. Aplikasi ini dikembangkan menggunakan **Kotlin** untuk membantu pengguna mendigitalkan dokumen fisik dengan cepat, rapi, dan berkualitas tinggi.

---

##  Fitur Utama

* **Pemindaian Cerdas:** Deteksi tepi dokumen secara otomatis dengan pilihan mode *Manual* atau *Ambil Otomatis*.
* **Pengaturan Kualitas:** Pilih kualitas gambar sesuai kebutuhan (**Low, Medium, High**).
* **Optimasi Dokumen:** Fitur **Auto Enhance** untuk mempertajam teks secara otomatis.
* **Manajemen PDF:** Simpan hasil pindaian ke format PDF dengan pilihan ukuran kertas (**A4** atau **Letter**).
* **Antarmuka Modern:** Mendukung **Dark Mode** untuk kenyamanan penggunaan di kondisi cahaya rendah.
* **Pangkas & Putar:** Sesuaikan sudut dokumen dengan presisi sebelum disimpan.
* **Scan Sejarah:** Kelola semua dokumen yang telah dipindai di satu tempat.

##  Teknologi & Tools

* **Bahasa Pemrograman:** Kotlin
* **Platform:** Android SDK
* **IDE:** Android Studio
